# Book_Management
자바 기반 책 관리 시스템

## 사용 언어 
- JAVA

## 데이터 구조
![Alt Text](http://dbscthumb.phinf.naver.net/4751_000_3/20170726205802619_YIY5S6NYA.png/992_170529_14542.png?type=w300_fst&wm=N)
- 이진탐색트리

## 기능
- 사용자 모드 - 검색
  - 책 이름, 저자, 출판사, 국가, 가격, 바코드, 발행년도로 검색 가능
  - 전체보기 가능
  - 상단 첫 행 속성 클릭으로 정렬가능
- 관리자 모드
  - 삽입, 삭제, 수정 기능
